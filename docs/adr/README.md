# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the ITOS project.

ADRs document significant architectural decisions — why they were made, what alternatives were considered, and what the expected consequences are. See [constitution/15-Engineering-Governance.md](../../constitution/15-Engineering-Governance.md) Section 5 for the ADR format and policy.

## Status Legend

| Status | Meaning |
